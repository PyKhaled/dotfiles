## Installation

This `dotfiles` repository is configured and managed by the [chezmoi](https://www.chezmoi.io/) project.

```shell
GITHUB_ORG=make-ops-tools
sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply $GITHUB_ORG
```
